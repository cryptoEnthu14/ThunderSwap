# ThunderSwap

ThunderSwap is the native decentralized exchange (DEX) built by the ThunderLaunch team.

## Overview

- **Graduation destination** for ThunderLaunch tokens — when a token completes its bonding curve, liquidity permanently migrates to ThunderSwap
- **General-purpose DEX** — any project can create a liquidity pool permissionlessly
- **Creator rewards** — token creators earn 0.15% on every post-graduation swap, claimable from ThunderLaunch or ThunderSwap directly
- **Multi-chain** — Solana v1, Base (L2) v2, Ethereum mainnet v3

## Status

Pre-development. Design and architecture phase in progress.
